# 어린이 공부방 (Kids Learning Room)

아이들을 위한 재미있고 상호작용적인 학습 앱입니다. 간단한 탭 인터랙션을 통해 영어 알파벳, 한글, 숫자, 그리고 다양한 동물들을 배울 수 있습니다.

🔗 **라이브 데모:** [https://taplingo.github.io](https://taplingo.github.io)

## 주요 기능

### 📚 학습 모드
- **영어 모드**:
  - **알파벳 이름**: A, B, C... 대문자와 소문자를 함께 배웁니다.
  - **알파벳 소리**: IPA 발음 기호를 보며 정확한 소리를 익힙니다.
  - **동물 공부 (NEW!)**:
    - **바다 동물**: 고래, 상어, 문어 등 19종의 바다 생물
    - **육지 동물**: 사자, 호랑이, 기린 등 19종의 육지 동물
    - **곤충**: 나비, 잠자리, 사마귀 등 20종의 곤충
    - 🎨 **특징**: 고품질 실사 일러스트와 영어 이름, 발음을 함께 제공합니다.
- **한글 모드**: 
  - 자음 이름 (기역, 니은, 디귿...)
  - 자음 소리 (그, 느, 드...)
  - 글자 (가, 나, 다...)
  - 모음 (ㅏ, ㅑ, ㅓ, ㅕ...)
  - 된소리 (ㄲ, ㄸ, ㅃ, ㅆ, ㅉ)
- **숫자 모드**:
  - 한국어 고유어 (하나, 둘, 셋...)
  - 한국어 한자어 (일, 이, 삼...)
  - 영어 (one, two, three...)
  - 자릿수 (1, 10, 100, 1000...)
  - 랜덤 숫자 (10~100)

### ✨ 특징
- **탭하여 학습**: 화면을 터치하면 다음 학습 내용이 랜덤/순차적으로 나옵니다.
- **반응형 이미지**: 모바일과 데스크탑, 태블릿 등 다양한 기기에서 최적화된 학습 환경을 제공합니다.
- **TTS 음성 지원**: 원어민 발음(영어/한국어)으로 읽어주는 기능 (켜기/끄기 가능).
- **중복 방지 알고리즘**: 같은 내용이 연속해서 나오지 않아 학습 효율을 높입니다.
- **아이 친화적 UI**: 직관적인 메뉴와 부드러운 애니메이션 효과.
## 시작하기

### 필수 요구사항

- Node.js 18+
- npm

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

로컬에서 실행 후 `http://localhost:5173` 접속

### 모바일에서 테스트

같은 WiFi에 연결된 상태에서:

```bash
npm run dev -- --host
```

표시되는 Network 주소 (예: `http://192.168.x.x:5173`)로 모바일 기기에서 접속

### 빌드 및 배포

```bash
npm run deploy
```
